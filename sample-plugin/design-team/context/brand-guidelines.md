# Brand Guidelines

## Voice
Clear, direct, and human. We avoid jargon and write as if explaining to a smart colleague.

## Tone by context
- **Onboarding**: Encouraging and guiding
- **Errors**: Calm, specific, never blame the user
- **Empty states**: Helpful and action-oriented

## Writing rules
- Use active voice
- Lead with the user benefit, not the feature
- Keep sentences under 20 words where possible
