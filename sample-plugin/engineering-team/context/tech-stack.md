# Tech Stack

## Backend
- Runtime: Node.js 22 (LTS)
- Framework: Express 4.x
- Database: PostgreSQL 16
- Cache: Redis 7
- Queue: BullMQ

## Frontend
- Framework: React 18 + TypeScript
- State: Zustand
- Styling: Tailwind CSS 3.x

## Infrastructure
- Cloud: AWS (ECS, RDS, ElastiCache)
- CI/CD: GitHub Actions
- Observability: Datadog
