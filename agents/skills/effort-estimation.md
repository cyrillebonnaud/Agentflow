# Effort Estimation

## What
Structured estimation of implementation effort, accounting for uncertainty, dependencies, and team factors.

## How
1. Break the work into independently estimable tasks
2. Estimate each task as a range (optimistic / realistic / pessimistic days)
3. Identify dependencies that could block or delay each task
4. Sum ranges and express as a confidence interval
5. Flag the top 3 risks that would push toward the pessimistic end
6. Separate effort (person-days) from calendar time (accounting for parallelism and team size)
