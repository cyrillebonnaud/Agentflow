# Heuristic Review

## What
A structured evaluation of a design artifact against established usability principles. Identifies friction, inconsistencies, and accessibility gaps before user testing.

## How
1. Evaluate against Nielsen's 10 heuristics — flag violations with severity (1-3)
2. Check WCAG 2.1 AA compliance for all interactive elements
3. Identify inconsistencies with the design system
4. Group findings by severity, not by heuristic
5. For each finding: what is wrong, why it matters, what to fix
