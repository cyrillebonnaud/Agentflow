# Accessibility Review

## What
Evaluation of a design artifact against WCAG 2.1 AA criteria, focusing on the four principles: Perceivable, Operable, Understandable, Robust.

## How
1. Check color contrast ratios (4.5:1 for normal text, 3:1 for large text and UI components)
2. Verify all interactive elements have visible focus states
3. Ensure all images and icons have meaningful alt text or are marked decorative
4. Check that form fields have associated labels
5. Verify keyboard navigation order is logical
6. Flag any content that relies on color alone to convey meaning
