# UX Direction

## What
A distinct design approach for a given problem. Each direction explores a different philosophy or trade-off — not just a visual variation.

## How
1. Name the direction with a descriptive slug in frontmatter (e.g. slug: minimalist-progressive)
2. State the core concept in one sentence
3. Describe the user journey through the key flows
4. Articulate 3-5 design decisions with rationale
5. Make trade-offs explicit: what this direction optimizes for vs. sacrifices
