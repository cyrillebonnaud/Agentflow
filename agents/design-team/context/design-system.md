# Design System

Our design system uses an 8px base grid. All spacing values are multiples of 8px.

## Tokens
- **Colors**: primary #2563EB, surface #F8FAFC, error #DC2626
- **Typography**: Inter — body 16px/1.5, heading 24-48px/1.2
- **Radius**: sm 4px, md 8px, lg 16px, pill 9999px
- **Shadows**: sm, md, lg elevation levels

## Components
Core components: Button (primary, secondary, ghost), Input, Select, Modal, Toast, Badge, Card, Table.

All components have documented states: default, hover, focus, active, disabled, error.
